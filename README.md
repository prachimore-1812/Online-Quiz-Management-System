🧠 Online Quiz Management System (SQL-Based)

📌 Project Overview

The "Online Quiz Management System" is a database-driven project designed to manage quiz-related data efficiently. It focuses on storing and organizing user (customer) information, enabling easy retrieval and analysis for future use. The system is implemented using SQL and executed in **MySQL Workbench**.

 🎯 Objectives

* To design a structured database for managing quiz participants
* To store and maintain customer/user information securely
* To demonstrate practical usage of SQL concepts in real-world scenarios

🛠️ Technologies Used

* SQL (Structured Query Language)
* MySQL Workbench

⚙️ Key Features

* 📊 Efficient storage of customer/user data
* 🔗 Implementation of JOIN clauses to combine data from multiple tables
* 🔍 Use of subqueries for complex data retrieval
* 🗂️ Organized relational database structure for better data management

🧩 Database Concepts Applied

* **Relational Schema Design**
* **Primary and Foreign Keys**
* **JOIN Operations** (INNER JOIN, etc.)
* **Subqueries** for nested data operations
* **Data Filtering and Retrieval**

🚀 How to Run the Project

1. Open MySQL Workbench
2. Import or create the database schema
3. Execute the SQL scripts provided in the project
4. Run queries to view, insert, and analyze quiz/user data.


📈 Use Case

This system can be used in educational platforms or online quiz applications to manage participants data, track records, and perform analysis using SQL queries.
